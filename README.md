# Databricks Intro Workshop for Microsoft Azure Government (MAG)

[![Databricks](https://img.shields.io/badge/Databricks-Solution_Accelerator-FF3621?style=for-the-badge&logo=databricks)](https://databricks.com)


## Intro
Welcome to the MAG Workshop, an introductory workshop designed for users working with Databricks on Microsoft Azure Government. This repository provides a comprehensive, easy-to-follow example to help you get started with Databricks in a secure, government-focused cloud environment.

## Overview
This workshop guides users through core Databricks concepts, cloud-native data analytics workflows, and hands-on exercises tailored for the Azure Government platform. The primary objective is to help users become comfortable with data engineering and analytics workloads in a compliant, secure setting using Databricks on Azure Government.

## Features
* End-to-End Tutorial: Step-by-step instructions covering Databricks basics and best practices.

* Databricks Notebooks: All workshop materials are provided as Databricks Notebooks for an interactive experience.

* Simple Setup: Users only need to update a **single configuration file** before starting the exercises.

## Getting Started

1. Ensure you have permissions to create a schema in your Databricks for Azure Government Environment and access to compute

2. Clone this Repository into your Databricks for Azure Government Environment
    ```bash
    git clone https://github.com/databricks-industry-solutions/mag-workshop.git
    ```

3. Update the Configuration File:
Before running the workshop notebooks, edit [1. Configure](./1.%20Configure.ipynb) (found in the root directory) to set a unique database/schema name from others.

4. Run the Notebooks:
After configuration, launch the individual Databricks Notebooks in your Databricks environment and follow the instructions for each module.
